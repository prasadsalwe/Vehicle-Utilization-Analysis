# Logistics Performance Dashboard

## Overview
The Vehicle Utilization Dashboard is a comprehensive PowerBI tool designed to analyze and visualize key metrics related to logistics operations, focusing on weight, volume, trip analysis and utilization of resources. This dashboard helps stakeholders make informed decisions to optimize **adhoc vehicle performance** as well.

## Technologies Used
-  PowerBi

## Features
- **Movement Analysis**: Compare performance between feeder and line haul movements.
- **Monthly Metrics**: View dedicated and adhoc master trips, actual weight utilization and volume utilization.
- **Trip Analysis**: Breakdown of trips by vehicle type and vendor showcasing the most efficient routes and vendors.
- **Utilization Metrics**: Detailed analysis of vehicle utilization rates and performance by destination.
- **Route Analysis**: Insights into specific routes, comparing actual weight and volume metrics.
- **Weekly Trends**: Visual representation of weekly performance metrics, allowing for trend analysis.
